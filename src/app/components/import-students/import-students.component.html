Excel File Format Requirements

Your Excel file should have the following columns in order:

  1. FirstName - Student's first name
  2. LastName - Student's last name
  3. Email - Valid email address
  4. CollegeName - Name of the college
  5. Department - Department name
  6. Major - Major/specialization
  7. GraduationDate - Date in DD-MM-YYYY format
  8. CGPA - CGPA on 0-10 scale
  9. CodilityTest1 - Score 0-100
  10. CodilityTest2 - Score 0-100
  11. CodilityTest3 - Score 0-100
  12. CodilityTest4 - Score 0-100
  13. CodilityTest5 - Score 0-100
  14. MidtermScore - Score on 0-4 scale
  15. GCScore - Score on 0-4 scale

Note: The first row should contain column headers exactly as listed above.
